Proyectos

Una aplicación web interactiva para gestionar tareas de manera eficiente. Con esta herramienta, los usuarios pueden agregar, editar, marcar como completadas o eliminar tareas de su lista de pendientes.
🛠 Tecnologías utilizadas:
✔️ HTML, CSS y JavaScript para la estructura, estilos y funcionalidad.
✔️ LocalStorage para guardar las tareas en el navegador y mantenerlas incluso después de recargar la página.
✔️ Diseño responsive para que se adapte a diferentes dispositivos.
